🎬 Movie Dataset Analysis

📌 Project Overview
This project analyzes a dataset of 3,000 movies with details such as budget, revenue, crew, cast, genres, and languages.
The goal is to help a rookie movie producer decide what type of movies to produce and which actors to cast, backed by thorough data-driven insights.

🛠️ Steps Taken
- Mounted Google Drive and imported the dataset into the notebook.
- Imported essential libraries: pandas, numpy, matplotlib, seaborn.
- Cleaned the dataset:
- Handled missing values in crew, cast, and genres.
- Replaced unrealistic budget/revenue values (like 0 or <1000) with median values.
- Created new columns for profit and ROI (Return on Investment).

🔎 Key Questions Answered
- Which movie made the highest profit?
- Identified the movie, its producer, director, and actors.
- Which language has the highest average ROI?
- Compared ROI across languages to find the most profitable language for production.
- What are the unique genres of movies in this dataset?
- Extracted and listed all distinct genres.
- Top producers by ROI
- Built a table of producers and directors for each movie.
- Found the top 3 producers with the highest average ROI.
- Most prolific actor
- Found the actor who acted in the most movies.
- Deep-dived into their movies, genres, and profits.
- Top 3 directors and their preferred actors
- Analyzed casting patterns to see which actors are most frequently chosen by the top directors.
